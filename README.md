\## Описание проекта

Random Password Generator — это графическое приложение на Python для генерации случайных паролей с гибкой настройкой параметров. Пользователь может выбрать длину пароля (от 6 до 32 символов) и наборы символов (цифры, строчные/прописные буквы, спецсимволы). Все сгенерированные пароли сохраняются в историю с деталями о параметрах генерации и времени.

\## Возможности

\- Настройка длины пароля с помощью ползунка.

\- Выбор используемых наборов символов через чекбоксы.

\- Генерация пароля по нажатию кнопки.

\- Отображение сгенерированного пароля в поле.

\- История последних 20 паролей с параметрами и временем.

\- Автоматическое сохранение и загрузка истории из `password\_history.json`.

\- Валидация: нельзя сгенерировать пароль без выбора символов.

\## Требования

\- Python 3.7+

\- Библиотека Tkinder (входит в стандартную поставку Python).

2\. Склонируй репозиторий:

&#x20;  cd random-password-generator

&#x20;  ```

3\. Запусти приложение:

&#x20;  ```bash

&#x20;  ```

4\. Для работы с Git используй стандартные команды (`git add .`, `git commit -m "..."`, `git push`).

\## Примеры использования

\- \*\*Сгенерировать пароль:\*\* выбери длину и наборы символов, нажми «Сгенерировать пароль».

\- \*\*История:\*\* последние 20 паролей отображаются в списке.

\- \*\*Закрыть и открыть:\*\* история сохраняется и загружается автоматически.

\- \*\*Тесты:\*\* проверь генерацию для минимальной (6) и максимальной (32) длины; попробуй снять все чекбоксы — должно быть предупреждение.
